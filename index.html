AWS Cloud Deployment

This website demonstrates a basic cloud deployment hosted on an AWS EC2 instance using Nginx. The deployment process is automated with GitHub Actions, allowing code changes to be securely delivered from the repository to the server with minimal manual effort.

Project Overview

The objective of this project is to understand the fundamentals of cloud hosting, automated deployments, and server configuration. It follows a simple CI/CD workflow where every successful code update is deployed to the EC2 instance through a secure automation process.

Technology Stack

  • Cloud Platform: AWS EC2
  • Web Server: Nginx
  • Automation: GitHub Actions
  • Deployment Script: Python (Paramiko)
  • Authentication: IAM & SSH Keys
